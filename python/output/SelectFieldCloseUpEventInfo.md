# 300.SelectFieldCloseUpEventInfo

300.1개요

300.2속성

```
SelectField 컴포넌트에서 화면에 표시된 아이템 리스트가 닫힐 때 발생하는 이벤트에 전달되는 EventInfo 오브젝트입니다.
```
