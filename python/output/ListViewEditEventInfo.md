# 278.ListViewEditEventInfo

278.1개요

278.2속성

```
ListView 컴포넌트의 Cell 에서 입력 컨트롤이 활성화 될 때 발생하는 이벤트에 전달되는 EventInfo 오브젝트입니다.
```
