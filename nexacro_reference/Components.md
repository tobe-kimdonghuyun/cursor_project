## Components > Animation Objects

### Animation Objects

> Components > Animation Objects

**Description**

컴포넌트에 애니메이션 효과를 줄 수 있는 오브젝트입니다.


---

## Components > Component

### Component

> Components > Component

**Description**

넥사크로에서 제공하는 컴포넌트 중 하위에 컴포넌트를 갖지 않는 기본 컴포넌트입니다.

**Remark**

- 일부 환경에서 지원되지 않는 컴포넌트가 있으므로 확인 후 사용하셔야 합니다.

## Components > ContainerComponent

### ContainerComponent

> Components > ContainerComponent

**Description**

넥사크로에서 제공하는 컴포넌트로서 하위에 다른 컴포넌트를 가질 수 있는 컨테이너 컴포넌트입니다.
