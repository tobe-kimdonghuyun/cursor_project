# 304.SketchEventInfo

304.1개요

304.2속성

```
Sketch 컴포넌트에 입력된 값을 value 속성 또는 파일로 저장에 성공했을 때 발생하는 이벤트에 전달되는 EventInfo 오브젝트입니다.
```
