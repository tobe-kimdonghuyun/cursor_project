# 317.TabMouseEventInfo

317.1개요

317.2속성

```
Tab 컴포넌트의 탭버튼에 표시된 Extra Button 을 클릭했을 때 발생하는 이벤트에 전달되는 EventInfo 오브젝트입니다.
```

| Mobile NRE | | Mobile WRE | | | | |
