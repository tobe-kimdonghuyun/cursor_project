# Nexacro Deploy JSON 결과물 설계

## 필드 설명
- schemaVersion: JSON 구조 버전. 나중에 1.1, 2.0으로 확장.
- tool.name, tool.version: 결과를 만든 도구 식별.
- execution.deployId: 실행 식별자. CI job ID와 매핑.
- execution.startedAt, execution.finishedAt: 실행 시작시간, 종료시간
- execution.status: `SUCCESS`, `FAILED`, `PARTIAL_SUCCESS`. 
- execution.scope: 배포 대상 범위(`project(기본)`, `file`, `service`, `module`)
- execution.durationMs: 실행 소요시간 (ms 또는 초)
- execution.exitCode: 쉘/프로세스 종료코드.
- project.name: 프로젝트명
- project.workspace: 프로젝트 경로
- project.nexacroVersion: nexacrolib 버전
- summary: 집계 정보.
    - message: 메시지
    - outputPath: 산출물 경로
    - artifactCount: 생성된 정상 산출물 개수
    - warningCount: 경고 개수
    - errorCount: 에러 개수
- artifacts[]: 배포 산출물 목록.
- warnings[], errors[]: warning / error 상세 목록.

## 상태값 상세
- SUCCESS: 정상 완료
- FAILED: 실패
- PARTIAL_SUCCESS: 일부 산출물 생성, 일부 실패
- CANCELLED: 외부 중단
